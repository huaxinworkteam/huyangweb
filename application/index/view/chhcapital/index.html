创合汇基金

创合汇基金是由教育机构、投资机构以及上海交通大学等国内外知名院校的校友共同发起设立，专注于投资优质的高成长型企业的私募股权投资基金。
创合汇基金凭借创合汇资本平台的优质资源，对所投项目除了提供资金支持外，还将提供专家咨询、人才资源、品牌宣传以及整合产业上下游资源等系统性支持。

创合汇投顾

  • 投融资服务

    凭借创合汇资本平台的专业优势及资源优势，根据企业的实际需要，为其提供融资顾问、投融资咨询、并购咨询、募投项目可行性研究、细分市场研究等服务。

  • 投行服务

    创合汇资本平台通过联合优质的券商、投行、律师事务所、会计师事务所等机构，对于有挂牌和上市潜质的企业提供全方位的挂牌和上市辅导。

  • 其他创业服务

    法律顾问服务、商标注册与知识产权服务、财务税务咨询服务、企划宣传服务、品牌营销服务、人才输送服务、产学研对接服务、创业孵化器
